# 📡 Wavecon 5G Performance Analysis

## 📌 Project Overview
This project analyzes the business impact of Wavecon's 5G rollout using data-driven insights. The goal is to evaluate revenue performance, customer behavior, and product portfolio changes post-launch.

---

## 🎯 Key Objectives
- Analyze revenue trends before and after 5G launch
- Identify customer churn and retention issues
- Evaluate plan-level performance (winners vs losers)
- Provide strategic business recommendations

---

## 📊 Tools Used
- Power BI (Dashboard & Visualization)
- SQL (Data Analysis)
- Excel (Data Cleaning)

---

## 📈 Key Insights

### 1. Revenue Paradox
- Revenue declined by 0.5%
- ARPU increased by 11%
👉 Higher value per user, but fewer users overall

### 2. Customer Retention Crisis
- TAU dropped by 8.28%
- Churn increased by 23.5%
👉 ~7M users lost in 4 months

### 3. Product Shift
- High-data plans (P1, P11) are growing
- Low-data & talktime plans declining

### 4. Critical Finding
👉 Forced migration led to customer loss instead of upgrades

---

## 🚀 Strategic Recommendations

1. Introduce 5G Lite Plan (affordable entry-level)
2. Focus on high-performing plans (P1, P11)
3. Improve network quality in high-churn cities
4. Avoid forced migration, use incentives
